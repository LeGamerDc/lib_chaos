## triangulation reduce pathfinding

mix of [TRA*](https://www.aaai.org/Papers/AAAI/2006/AAAI06-148.pdf) and [Polyanya](https://www.ijcai.org/proceedings/2017/0070.pdf) 
algorithm to create currently most advanced pathfinding lib which is optimal while fast. [further reading](http://www.aamircheema.com/research/ESPP_AIJ2021.pdf)
